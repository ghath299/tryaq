إعادة تصميم واجهة المستخدم (UI/UX) بشكل كامل مع التركيز الشديد على الأنيميشن.
ممنوع تغيير أي منطق أو باكند أو ميزات.
التعديل بصري وحركي فقط.

====================================
الهوية اللونية (إلزامي)
====================================
- اللون الأساسي: سماوي فاتح (#5EDFFF أو قريب منه)
- اللون الثانوي: أزرق هادئ (#1F6AE1 أو قريب)
- اللون الخلفي: أبيض نقي
- لا تستخدم ألوان داكنة أو سوداء في الواجهة الافتراضية
- Dark Mode يكون اختياري لاحقًا

====================================
الإحساس العام للتطبيق
====================================
- طبي
- نظيف
- حديث
- فاخر
- يشعر المستخدم أنه تطبيق 2025
- لا يبدو تجريبي أو بدائي

====================================
قواعد صارمة
====================================
- ممنوع الأزرار المربعة
- ممنوع الكروت التقليدية
- ممنوع واجهات ثابتة
- كل عنصر يجب أن يتحرك أو يتفاعل

====================================
الأنيميشن (أهم جزء)
====================================
- كل انتقال بين شاشة وأخرى يكون بانيميشن ناعم (Fade + Slide)
- Bottom Navigation:
  - الأيقونة تتحرك عند الاختيار
  - تأثير موجة أو توسّع خفيف
- الأزرار:
  - Scale خفيف عند الضغط
  - Ripple ناعم بلون سماوي
- القوائم:
  - تظهر بانزلاق ناعم
  - مع Blur للخلفية
- العناصر:
  - تظهر تدريجيًا عند التمرير (Scroll animation)
- لا يوجد أي عنصر يظهر فجأة بدون حركة

====================================
واجهة تسجيل الدخول
====================================
- خلفية متدرجة (سماوي → أبيض)
- الحقول عائمة (Floating)
- زر تسجيل متحرك بنبض خفيف
- إدخال الاسم ورقم الهاتف يكون بانيميشن سلس

====================================
الواجهة الرئيسية (Home)
====================================
- Hero Section متحرك
- سلايدر بانيميشن تلقائي
- أقسام تطفو (Floating Sections)
- بطاقات الأطباء والصيدليات:
  - حواف دائرية
  - ظل ناعم
  - حركة عند الظهور
  - حركة عند الضغط

====================================
الأطباء والصيدليات
====================================
- صور داخل دوائر ناعمة
- مؤشرات المسافة والموقع بشكل Chips
- الانتقال لصفحة التفاصيل يكون بانيميشن انسيابي

====================================
قسم العلاجات
====================================
- شريط بحث بتوهج سماوي
- قسم الذكاء الاصطناعي بتصميم مستقبلي
- رفع الصورة يكون مثل Scan ذكي وليس Upload تقليدي

====================================
القائمة الجانبية ☰
====================================
- خلفية مع Blur
- تنزلق بسلاسة
- العناصر تظهر واحدة تلو الأخرى بانيميشن
- لا تظهر فجأة

====================================
الخطوط
====================================
- خط عربي حديث وواضح
- عناوين كبيرة
- نصوص ثانوية خفيفة

====================================
الخلاصة
====================================
أعد تصميم التطبيق ليكون:
- سماوي / أزرق / أبيض
- مليء بالأنيميشن
- ناعم جدًا
- احترافي
- بدون أي مظهر بدائي

نفّذ الآن إعادة التصميم بالكامل.
